Features
为什么选择 Liquid Glass
不依赖任何实验性 API，完全基于 W3C 标准的 SVG Filter 规范，实现了苹果风格的液态玻璃折射效果。
🎯

实时 DOM 折射

不是模拟、不是截图——feDisplacementMap 直接作用于渲染中的 DOM 内容，实时折射页面上的任何元素。

⚡

Canvas 像素级位移

逐像素计算球面折射强度，smoothstep 插值 + Curvature 曲率控制，精确模拟光学透镜。

🌈

Chroma 色散

Chroma 参数控制镜头边缘棱镜色散，Splay 控制向外扩散强度，模拟真实光学色差。

🌍

全浏览器原生支持

Chrome、Safari、Firefox 桌面端和移动端全部支持，零 Feature Flag，无需回退方案。

🎬

图片 & 视频支持

镜头可以折射静态图片，也可以折射正在播放的视频——因为滤镜作用于 DOM 而非像素缓冲区。

🎛

10 维参数可调

Size / Chroma / Depth / Blur / Edge Highlight / Strength / Curvature / Splay / Glow / Specular Angle——专业级控制。

Code
核心实现
只需 SVG Filter + Canvas 位移贴图，即可在任何 React 项目中集成液态玻璃效果。
// Canvas 位移贴图 — Curvature 控制折射衰减曲线 const curvExp = 1 + curvature * 3 // 1(线性) → 4(强球面) const refraction = Math.pow(1 - smoothT, curvExp) data[idx] = 128 + (-nx * refraction * depth * 1.6) // R → X data[idx+1] = 128 + (-ny * refraction * depth * 1.6) // G → Y // SVG Filter — Strength 映射到 displacement scale <feDisplacementMap scale="{strength * 1000}" /> // Lens 视觉 — Specular Angle 控制高光方向 const hlX = 50 + 30 * Math.sin(angle) const hlY = 50 - 30 * Math.cos(angle)
