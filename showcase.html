Features
为什么选择 Liquid Glass
不依赖任何实验性 API，完全基于 W3C 标准的 SVG Filter 规范，实现了苹果风格的液态玻璃折射效果。
🎯

实时 DOM 折射

不是模拟、不是截图——feDisplacementMap 直接作用于渲染中的 DOM 内容，实时折射页面上的任何元素。

⚡

Canvas 像素级位移

逐像素计算球面折射强度，smoothstep 插值 + 二次衰减，精确模拟光学透镜的放大效果。

🌈

边缘色散

镜头边缘模拟棱镜色散，红/绿通道独立偏移产生微妙的彩虹折射效果。

🌍

全浏览器原生支持

Chrome、Safari、Firefox 桌面端和移动端全部支持，零 Feature Flag，无需回退方案。

🎬

图片 & 视频支持

镜头可以折射静态图片，也可以折射正在播放的视频——因为滤镜作用于 DOM 而非像素缓冲区。

🎛

全参数可调

镜头半径、位移强度、折射深度、边缘宽度、色散强度——所有参数实时调节，即时预览。

Code
核心实现
只需 SVG Filter + Canvas 位移贴图，即可在任何 React 项目中集成液态玻璃效果。
// 1. Canvas 生成位移贴图 (R=X偏移, G=Y偏移) const imageData = ctx.createImageData(w, h) for (let y = 0; y < h; y++) { for (let x = 0; x < w; x++) { const dist = Math.sqrt(dx*dx + dy*dy) const strength = (1 - smoothT) * (1 - smoothT) data[idx] = 128 + (-nx * strength * cap) // R → X data[idx + 1] = 128 + (-ny * strength * cap) // G → Y } } // 2. SVG Filter 消费位移贴图 <filter id="liquid-glass"> <feImage href="{canvas.toDataURL()}" /> <feDisplacementMap in="SourceGraphic" in2="displacementMap" scale="55" xChannelSelector="R" yChannelSelector="G" /> </filter> // 3. 应用到任意 DOM 元素 <div style={{ filter: 'url(#liquid-glass)' }}> {children} </div>
